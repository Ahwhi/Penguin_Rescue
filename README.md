### 🧾 피드백 및 개선 과정

1. 처음 시작 시, 세이브포인트 바로 앞에 벽이 생성될 경우 
피하기가 어렵다.

2. 조작이 상당히 불편하게 느껴졌습니다. 초반부에는 비교적 
수월하게 인질을 구출할 수 있으면 좋을것 같습니다.

3. 장애물이 랜덤으로 생성되는 것 같은데 무조건 장애물에 
부딪힐 수 밖에 없는 부분이 나오는 것이 아쉽다.  

4. 시작 부분에는 오브젝트 배치를 줄여 플레이어가 조작을 
준비할 시간을 주는 편이 좋을 것 같다. 

5. 맵이 계속 반복되는 구조이고, 경우에 따라 무조건 HP를 잃는 
구간이 있는데 이 부분을 개선했으면 좋겠다.

6. 체력을 회복 할수 있는 아이템을 추가하거나 장애물의 배치를 
쉽게 바꿀 필요가 있어보임.  

7. 위 쪽 방향 키로 하고, 로프 탈착을 스페이스바로 하면 
플레이에 더 편할 것 같습니다. 

8. 왼쪽 키와 오른쪽 키를 동시에 누르면 멈추지 않고, 속도가 
낮아져서 로프를 타지 않고 벽을 피할 수 있습니다. 왼쪽 키와 
오른쪽 키가 동시에 눌렸다면, 속도를 0으로 설정하면 좋을 것 
같습니다.

● 개선점

1. 시작 부분에 오브젝트 배치를 줄이고 발판 길이를 늘림으로써 
플레이어의 조작 준비를 돕고 초반 난이도를 낮춤 

2. 중력을 상승시켜 점프의 횟수를 늘리고, 그만큼 유저의 예측 
점프 비율이 줄어 난이도를 낮춤 

3. 체력 회복 아이템을 추가하여 난이도를 낮춤 

4. 왼쪽 키와 오른쪽 키의 속도 조절값을 같게 하여 조작에 대한 
균형을 맞추어 유저 친화적으로 변경 

5. 이단 점프를 추가하여 더욱 흥미롭고 안전한 플레이 가능 

6. 조작 빈도가 많은 점프를 스페이스에서 위쪽 방향키로 
변경하여 유저의 불편함을 제거
